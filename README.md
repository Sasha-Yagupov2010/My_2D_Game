# Capture the Flag Game

## Описание
Простая 2D игра в жанре "Capture the Flag", разработанная с использованием библиотеки SFML. 
## Управление

### Режим двух игроков:
**Игрок 1 (Красный):**
- Движение: W, A, S, D
- Захват флага: E
- Стрельба: Q

**Игрок 2 (Синий):**
- Движение: I, J, K, L
- Захват флага: O
- Стрельба: U

### Общие клавиши:
- Выход в меню: ESC
- Продолжить после победы: Space


### Зависимости:
- **SFML 2.5+** (Graphics, Window, System)
- **Компилятор C++** с поддержкой C++11
- **Windows** (используются WinAPI функции для поиска файлов)


## Установка и сборка

### 1. Установка SFML
1. Скачайте SFML с официального сайта: https://www.sfml-dev.org/
2. Установите библиотеку или добавьте в переменные среды

### 2. Сборка проекта
```bash
# Клонирование проекта (если используется Git)
git clone <repository-url>
cd <project-directory>

# Сборка с помощью CMake
mkdir build
cd build
cmake ..
cmake --build .

# Или прямая компиляция (пример для g++)
g++ -o game main.cpp -lsfml-graphics -lsfml-window -lsfml-system
```

### 3. Структура проекта
```
game/
├── main.cpp              # Основной файл игры
├── libs/                 # Пользовательские библиотеки
│   ├── Settings.h
│   ├── Player.h
│   ├── Mytext.h
│   ├── MyButton.h
│   ├── GameMap.h
│   ├── Flag.h
│   └── ShootGun.h
├── maps/                 # Директория с картами
│   ├── fastrunner.txt   # Карта по умолчанию
```

## Создание карт

### Формат карт:
Карты хранятся в текстовых файлах с расширением `.txt` в папке `maps/`. Размер карты фиксированный: 24×16 тайлов.

### Символы:
- `1` - стена/препятствие
- `0` или пробел - проходимая область
- Другие символы игнорируются


## Настройки игры

### Конфигурация:
Настройки можно изменить в файле `libs/Settings.h` или непосредственно в коде `main.cpp`:
- Размер окна (ширина и высота)
- FPS (кадров в секунду)
- Минимальное расстояние для захвата флага
- Количество очков для победы

## Геймплей

### Цель игры:
Захватить флаг противника и доставить его на свою базу. Игрок побеждает, когда набирает установленное количество очков.
